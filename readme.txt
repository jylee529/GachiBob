1. 프로젝트의 기능과 목적
    * 개최자는 식사 모임을 시공간적 정보와 제공할 음식, 모집 인원 수 등을 제공하고,
    참여자는 등록된 정보를 지도를 바탕으로 범위 탐색을 할 수 있다.

    * 이로써 인근의 이웃과 식사 모임을 진행하기가 훨씬 수월해질 것이다.

2. 사용 방법
    a. 이메일로 회원가입을 한다.
    b. 로그인을 한다.
    c. 식사 모임을 등록하거나 검색을 통해 식사 모임에 대한 정보를 얻는다.
        c - 1. 등록을 한다면 모임 장소(도로명 주소), 모임 날짜, 모임 시간, 제공할 식사 메뉴, 모집 인원 등을 입력한다.
            주소가 실제로 존재하는 주소면 등록에 성공한다. 위에 상기한 정보에 더해서 등록자의 이메일이 함께 등록된다.
        c - 2. 검색은 원하는 주소와 탐색 반경(1km, 5km, 10km)을 설정하면 그 주소를 중심으로 탐색 반경 안에 속하는 
            식사 모임들이 우측에는 지도 상에 위치가 표시되고 하단에는 리스트로 등록 정보가 나온다.
    d. 검색한 정보를 바탕으로 원하는 식사 모임에 이메일로 등록자에게 연락을 하여 모임을 진행하면 된다. 

3. 개발 환경
    * 언어 : HTML, CSS, JavaScript
    * IDE : VSCode
    * Database : 계정 정보 - firebase authentification, 사용자 등록 정보 : firestore Database
    * API : firebase, kakaoMap
    * 서버 : 깃허브(현재 지도 api 작동 안함. 업데이트가 너무 느려서 수정과 점검을 위해 로컬로만 설정해놓은 상태.)
